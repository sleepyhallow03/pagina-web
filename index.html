Introducción a la Programación

Bienvenidos al Recurso Educativo Abierto (REA) sobre Conceptos Básicos de Programación. Este material está diseñado para estudiantes de noveno grado que deseen aprender los fundamentos de la programación.

La programación es el proceso de crear instrucciones para que las computadoras realicen tareas específicas. En esta unidad aprenderás:

  • Qué son los algoritmos y cómo se usan para resolver problemas
  • Diferentes formas de expresar algoritmos (diagramas de flujo, pseudocódigo)
  • Qué son los lenguajes de programación y sus tipos
  • Conceptos básicos como variables, condicionales y bucles

Contenidos Específicos

1. Algoritmos

Un algoritmo es un conjunto ordenado de instrucciones para resolver un problema. Es como una receta que le dice a la computadora qué pasos seguir.

Ejemplo: Algoritmo para sumar dos números:

  1. Pedir el primer número
  2. Pedir el segundo número
  3. Sumar los dos números
  4. Mostrar el resultado

2. Formas de Expresión de Algoritmos

Podemos representar algoritmos de diferentes maneras:

Diagramas de Flujo

Usan símbolos gráficos para representar los pasos de un algoritmo.

Pseudocódigo

Es una descripción informal de alto nivel de un algoritmo que usa una mezcla de lenguaje natural y estructuras de programación.

INICIO
    LEER numero1
    LEER numero2
    resultado = numero1 + numero2
    ESCRIBIR resultado
FIN

3. Lenguajes de Programación

Un lenguaje de programación es un sistema de comunicación que permite a los humanos dar instrucciones a las computadoras.

Tipos de lenguajes:

  • Lenguajes de alto nivel: Más cercanos al lenguaje humano (Python, Java, JavaScript)
  • Lenguajes de bajo nivel: Más cercanos al lenguaje máquina (Ensamblador)
  • Lenguajes compilados: Se traducen completamente antes de ejecutarse (C++)
  • Lenguajes interpretados: Se traducen línea por línea durante la ejecución (Python, JavaScript)

Ejercicios Prácticos

Ejercicio 1: Crear un algoritmo

Crea un algoritmo en pseudocódigo para calcular el área de un rectángulo. Recuerda que el área se calcula multiplicando la base por la altura.

Ejercicio 2: Identificar tipos de lenguajes

Clasifica los siguientes lenguajes según su tipo (alto/bajo nivel, compilado/interpretado):

  • Python
  • Ensamblador
  • C++
  • JavaScript
